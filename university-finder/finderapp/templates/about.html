What is University Finder?

University Finder is a web app that allows users to search for universities by country using public data from the Hipolabs Universities API.

Why I Built This

I created this project to learn Flask and build my skills in web development by working with real-world APIs, forms, and templates.

Tech Stack

  • Python & Flask for backend
  • HTML & CSS for frontend
  • Hipolabs Universities API for data

Future Improvements

  • Filter by domain or study field
  • Clickable links to university websites
  • Mobile-friendly layout
